# Privacy Boundary

This project is intentionally generic.

## Allowed content

- Generic Ubuntu commands.
- Public Linux tool behavior.
- Artificial examples.
- Sanitized test fixtures.
- Generic parser logic.

## Disallowed content

- Company names from private work contexts.
- Customer names.
- Real server serial numbers.
- Real MAC addresses.
- Real IP addresses.
- Internal project names.
- Private platform topology.
- Proprietary firmware versions.
- Internal validation SOPs.
- NDA-protected material.

## Review checklist before publishing logs

- Search for serial-like strings.
- Search for MAC address patterns.
- Search for IPv4 addresses.
- Search for internal hostnames.
- Search for customer and company names.
- Search for asset tags.
